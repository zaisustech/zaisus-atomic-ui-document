Getting Started with Atomic UI

January 15, 2026 Â· One min read
Core Developer

A comprehensive guide to help you get started with Atomic UI in your React Native project.

Installationâ€‹

First, you'll need to set up authentication since Atomic UI is a private package hosted on GitHub Packages. Follow our installation guide for detailed steps.

Basic Setupâ€‹

Once installed, wrap your app with the required providers:

import { ThemeProvider, ToastProvider } from '@zaisustech/atomic-ui';

function App() {
  return (
    <ThemeProvider>
      <ToastProvider>
        {/* Your app content */}
      </ToastProvider>
    </ThemeProvider>
  );
}

Your First Componentâ€‹

Let's create a simple example using a few components:

import { Button, Text, Card } from '@zaisustech/atomic-ui';

function MyScreen() {
  return (
    <View>
      <Text variant="Header.large.bold">My App</Text>
      <Card variant="medium">
        <Text variant="Paragraph.default.regular">
          This is a card component with medium shadow.
        </Text>
        <Button 
          title="Click Me" 
          variant="primary"
          onPress={() => console.log('Button pressed!')}
        />
      </Card>
    </View>
  );
}
