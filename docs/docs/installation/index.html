Installation

Prerequisites​

  • Node.js 18+ or Bun
  • React Native 0.76+
  • Expo SDK 52+ (required for @zaisustech/file-system)

Install packages​

Install only the packages you need:

# All packages (Bun)
bun add @zaisustech/atomic-ui @zaisustech/file-system @zaisustech/drawing

# All packages (npm)
npm install @zaisustech/atomic-ui @zaisustech/file-system @zaisustech/drawing

# All packages (yarn)
yarn add @zaisustech/atomic-ui @zaisustech/file-system @zaisustech/drawing

Per-package​

# Atomic UI only
bun add @zaisustech/atomic-ui

# File System only
bun add @zaisustech/file-system

# Drawing only
bun add @zaisustech/drawing

Peer dependencies​

Install peer dependencies in your app. If you use Expo, many are already present.

@zaisustech/atomic-ui​

PackagePurpose
react^18.3.1
react-native^0.76.5

@zaisustech/file-system​

PackagePurpose
@zaisustech/atomic-uiUI components (Card, Icon, Text, Dialog, Toast, etc.)
expoExpo runtime
expo-file-systemFile system API
expo-sharingShare files (optional, for open/share flows)
react-native-gesture-handlerGestures (swipe actions)
react-native-reanimatedAnimations
react-native-keyboard-controllerKeyboard-aware UI (optional)

The file-system package also depends on date-fns (bundled) for date formatting (e.g. "Today 02:30 AM", "Yesterday 03:20 PM").

@zaisustech/drawing​

PackagePurpose
react-native-svgSVG rendering for drawing paths
react-native-view-shotCapture drawing as image

One-line install (all peers for file-system + drawing)​

bun add @zaisustech/atomic-ui expo expo-file-system expo-sharing react react-native react-native-gesture-handler react-native-reanimated react-native-keyboard-controller react-native-svg react-native-view-shot

Adjust for npm/yarn and omit packages you already have.

Verify​

import { Button } from '@zaisustech/atomic-ui';
import { FileManagerProvider, FileManagerContainer } from '@zaisustech/file-system';
import { SignaturePad } from '@zaisustech/drawing';

If these imports resolve without error, installation is correct.
