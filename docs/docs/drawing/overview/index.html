Drawing Overview

@zaisustech/drawing is a React Native drawing and signature pad component built with Expo. Capture signatures and drawings using SVG rendering with customizable pen styles and export capabilities.

Features​

  • Signature Pad: Draw signatures and freeform drawings
  • Pen & Eraser: Switch between pen and eraser modes; eraser draws with background color for a natural erase effect
  • Undo / Redo: Remove or restore the last stroke(s) via ref methods
  • Customizable Styles: Configure pen color, size, eraser size, line caps, and joins
  • Export: Capture drawings as images (PNG, JPG, WebM, RAW)
  • SVG Rendering: Smooth, scalable vector graphics
  • Touch Gestures: Natural drawing experience with pan responder

Installation​

# Bun
bun add @zaisustech/drawing

# npm
npm install @zaisustech/drawing

# yarn
yarn add @zaisustech/drawing

Peer dependencies​

Install these in your app if not already present:

PackagePurpose
react-native-svgSVG rendering for drawing paths

bun add react-native-svg react-native-view-shot

Basic usage​

Use SignaturePad with a ref to control clearing and capturing:

import React, { useRef } from 'react';
import { View, Button } from 'react-native';
import { SignaturePad, SignaturePadRef } from '@zaisustech/drawing';

const App = () => {
  const signatureRef = useRef<SignaturePadRef>(null);

  const handleSave = () => {
    signatureRef.current?.readSignature();
  };

  const handleClear = () => {
    signatureRef.current?.clearSignature();
  };

  const handleUndo = () => {
    signatureRef.current?.undo();
  };

  const handleRedo = () => {
    signatureRef.current?.redo();
  };

  return (
    <View style={{ flex: 1 }}>
      <SignaturePad
        ref={signatureRef}
        penColor="#000000"
        penSize={3}
        backgroundColor="#FFFFFF"
        onSignatureOK={(dataUri) => {
          console.log('Signature captured:', dataUri);
        }}
      />
      <Button title="Undo" onPress={handleUndo} />
      <Button title="Redo" onPress={handleRedo} />
      <Button title="Save" onPress={handleSave} />
      <Button title="Clear" onPress={handleClear} />
    </View>
  );
};

Customization​

Pen and Eraser Modes​

Switch between drawing and erasing with the mode prop. In eraser mode, strokes use the background color and eraserSize so they appear as erasing.

import type { SignaturePadMode } from '@zaisustech/drawing';

const [mode, setMode] = useState<SignaturePadMode>('pen');

<SignaturePad
  mode={mode}           // 'pen' | 'eraser'
  eraserSize={24}       // Eraser stroke width (default 24)
  penColor="#000000"
  penSize={3}
  backgroundColor="#FFFFFF"
/>
// Toggle with: setMode('pen') or setMode('eraser')

Pen Styles​

<SignaturePad
  penColor="#FF0000"        // Red pen
  penSize={5}               // Thicker strokes
  strokeLinecap="round"     // Rounded line ends
  strokeLinejoin="round"    // Rounded line joins
/>

Background​

<SignaturePad
  backgroundColor="#F5F5F5"  // Light gray background
/>

Export Format​

<SignaturePad
  format="png"  // Options: 'jpg' | 'png' | 'webm' | 'raw'
  onSignatureOK={(dataUri) => {
    // dataUri contains the captured image
  }}
/>

Callbacks​

The component provides several callbacks for tracking drawing state:

<SignaturePad
  onBegin={() => {
    console.log('Drawing started');
  }}
  onEnd={() => {
    console.log('Drawing ended');
  }}
  onClear={() => {
    console.log('Signature cleared');
  }}
  onSignatureOK={(dataUri) => {
    console.log('Signature captured:', dataUri);
    // Save or process the data URI (e.g. upload or save to file)
  }}
/>
