Drawing API Reference

Reference for @zaisustech/drawing aligned with libs/drawing. Exports include the SignaturePad component and types: SignaturePadRef, SignaturePadProps, SignaturePadMode, StrokePath.

Components​

SignaturePad​

Main component for capturing signatures and drawings.

Props​

PropTypeDefaultDescription
penColorstring'#000000'Color of the pen stroke
penSizenumber3Width of the pen stroke in pixels
mode'pen' | 'eraser''pen'Drawing mode: pen draws with pen color/size; eraser draws with background color and eraserSize
eraserSizenumber24Width of the eraser stroke in pixels (used when mode === 'eraser')
backgroundColorstring'#FFFFFF'Background color of the pad
strokeLinecap'butt' | 'round' | 'square''round'Style of line endings
strokeLinejoin'miter' | 'round' | 'bevel''round'Style of line joins
fillstring'none'Fill color for paths (usually 'none' for strokes)
format'jpg' | 'png' | 'webm' | 'raw''png'Export format when capturing
onBegin() => void—Callback invoked when drawing begins
onEnd() => void—Callback invoked when drawing ends
onClear() => void—Callback invoked when signature is cleared
onSignatureOK(dataUri: string) => void—Callback with captured signature as data URI
childrenReact.ReactNode—Optional children to render behind the drawing layer

Methods (via ref)​

Access methods through a ref:

const signatureRef = useRef<SignaturePadRef>(null);

// Clear all drawn paths
signatureRef.current?.clearSignature();

// Undo the last stroke
signatureRef.current?.undo();

// Redo the last undone stroke
signatureRef.current?.redo();

// Capture the current drawing as an image
signatureRef.current?.readSignature();
MethodDescription
clearSignature()Clears all drawn paths and resets the pad (and redo stack)
undo()Removes the last committed stroke; it can be restored with redo()
redo()Restores the last undone stroke
readSignature()Captures the current drawing as an image and calls onSignatureOK with the data URI

Types​

SignaturePadMode​

type SignaturePadMode = 'pen' | 'eraser';

SignaturePadRef​

type SignaturePadRef = {
  clearSignature: () => void;
  readSignature: () => void;
  undo: () => void;
  redo: () => void;
};

SignaturePadProps​

type SignaturePadProps = {
  children?: React.ReactNode;
  penColor?: string;
  penSize?: number;
  mode?: SignaturePadMode;
  eraserSize?: number;
  backgroundColor?: string;
  strokeLinecap?: 'butt' | 'round' | 'square';
  strokeLinejoin?: 'miter' | 'round' | 'bevel';
  fill?: string;
  format?: 'jpg' | 'png' | 'webm' | 'raw';
  onBegin?: () => void;
  onEnd?: () => void;
  onClear?: () => void;
  onSignatureOK?: (dataUri: string) => void;
};

StrokePath​

Internal type representing a single stroke path:

type StrokePath = {
  d: string;           // SVG path data
  color: string;        // Stroke color
  strokeWidth: number; // Stroke width
};

Examples​

Basic Signature Capture​

import React, { useRef } from 'react';
import { View, Button } from 'react-native';
import { SignaturePad, SignaturePadRef } from '@zaisustech/drawing';

const SignatureScreen = () => {
  const signatureRef = useRef<SignaturePadRef>(null);

  const handleSave = () => {
    signatureRef.current?.readSignature();
  };

  return (
    <View style={{ flex: 1 }}>
      <SignaturePad
        ref={signatureRef}
        onSignatureOK={(dataUri) => {
          // Save or upload the data URI
          console.log('Signature:', dataUri);
        }}
      />
      <Button title="Save Signature" onPress={handleSave} />
    </View>
  );
};

Pen, Eraser, Undo, and Redo​

import { SignaturePad, type SignaturePadMode, type SignaturePadRef } from '@zaisustech/drawing';

const [mode, setMode] = useState<SignaturePadMode>('pen');

<>
  <View style={{ flexDirection: 'row', gap: 8 }}>
    <Button title="Pen" onPress={() => setMode('pen')} />
    <Button title="Eraser" onPress={() => setMode('eraser')} />
    <Button title="Undo" onPress={() => signatureRef.current?.undo()} />
    <Button title="Redo" onPress={() => signatureRef.current?.redo()} />
  </View>
  <SignaturePad
    ref={signatureRef}
    mode={mode}
    eraserSize={24}
    penColor="#000000"
    penSize={3}
    backgroundColor="#FFFFFF"
    onSignatureOK={(dataUri) => { /* ... */ }}
  />
</>
