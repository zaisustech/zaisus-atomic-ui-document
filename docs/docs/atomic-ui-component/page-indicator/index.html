PageIndicator

A visual indicator component for displaying pagination dots that show the current page position and allow navigation.

Category: Molecule
Dependencies: View

Interactive Preview​

Edit the code to customize the PageIndicator:

Usage​

import { PageIndicator } from '@zaisustech/atomic-ui';
import { useState } from 'react';

const MyComponent = () => {
  const [currentPage, setCurrentPage] = useState(0);

  return (
    <PageIndicator
      count={5}
      currentPage={currentPage}
      onPageChange={setCurrentPage}
      variant="primary"
    />
  );
};

Props​

count​

Type: number

Required. Total number of pages/indicators to display.

currentPage​

Type: number

Required. Currently active page index (0-based).

variant​

Type: 'default' | 'primary' | 'secondary'

Default value: 'default'

Visual style variant. Controls the color of the active and inactive dots.

  • 'default' - Neutral gray color variant
  • 'primary' - Primary brand color variant
  • 'secondary' - Secondary brand color variant

size​

Type: number

Default value: 8

Size of the inactive indicator dots in pixels. The active dot is automatically 1.5x the size of the inactive dots.

gap​

Type: number

Default value: spacing.xs (from theme)

Gap between indicator dots in pixels. If not provided, uses the theme's small spacing value.

onPageChange​

Type: (index: number) => void

Optional callback function that is called when a dot is pressed. The function receives the index of the pressed dot. If not provided, dots are not interactive.

style​

Type: StyleProp<ViewStyle>

Custom styles for the container.

defaultStyle​

Type: StyleProp<ViewStyle>

Default value: undefined

Custom styles applied to all indicator dots (both active and inactive). This style is overridden by activeStyle for the active dot.

activeStyle​

Type: StyleProp<ViewStyle>

Default value: undefined

Custom styles applied specifically to the active indicator dot. This style is applied in addition to defaultStyle and will override conflicting properties.

Examples​

Basic PageIndicator​

import { PageIndicator } from '@zaisustech/atomic-ui';

<PageIndicator count={3} currentPage={0} />

With Navigation​

import { PageIndicator } from '@zaisustech/atomic-ui';
import { useState } from 'react';

function MyComponent() {
  const [currentPage, setCurrentPage] = useState(0);

  return (
    <PageIndicator
      count={5}
      currentPage={currentPage}
      onPageChange={setCurrentPage}
    />
  );
}

Variants​

<PageIndicator count={3} currentPage={0} variant="default" />
<PageIndicator count={3} currentPage={1} variant="primary" />
<PageIndicator count={3} currentPage={2} variant="secondary" />

Sizes​

<PageIndicator count={3} currentPage={0} size={6} />
<PageIndicator count={3} currentPage={1} size={8} />
<PageIndicator count={3} currentPage={2} size={12} />

With Custom Gap​

<PageIndicator count={3} currentPage={0} size={8} gap={12} />
<PageIndicator count={3} currentPage={1} size={10} gap={16} />

With Custom Styles (Border + Filled)​

import { PageIndicator } from '@zaisustech/atomic-ui';
import { StyleSheet } from 'react-native';

const styles = StyleSheet.create({
  defaultDot: {
    borderWidth: 1,
    borderColor: '#007AFF',
    backgroundColor: 'transparent',
  },
  activeDot: {
    backgroundColor: '#007AFF',
    borderWidth: 0,
  },
});

<PageIndicator
  count={5}
  currentPage={2}
  defaultStyle={styles.defaultDot}
  activeStyle={styles.activeDot}
/>

This example shows:

  • Inactive dots: Circles with border only (transparent background)
  • Active dot: Filled circle (solid background, no border)

All Variants​

With Pager Component​

import { PageIndicator, Pager } from '@zaisustech/atomic-ui';
import { useState } from 'react';

function MyComponent() {
  const [currentPage, setCurrentPage] = useState(0);

  return (
    <>
      <Pager page={currentPage} onPageChange={setCurrentPage}>
        <View><Text>Page 1</Text></View>
        <View><Text>Page 2</Text></View>
        <View><Text>Page 3</Text></View>
      </Pager>
      <PageIndicator
        count={3}
        currentPage={currentPage}
        onPageChange={setCurrentPage}
        variant="primary"
      />
    </>
  );
}
