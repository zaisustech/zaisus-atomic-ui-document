Dialog

A modal dialog component for displaying confirmations, alerts, and custom content.

Category: Organism
Dependencies: DialogButton, Icon, Text, Modal

Interactive Preview​

Edit the code to customize the Dialog:

Usage​

import { Dialog } from '@zaisustech/atomic-ui';
import { useState } from 'react';

const MyComponent = () => {
  const [visible, setVisible] = useState(false);

  return (
    <>
      <Button title="Show Dialog" onPress={() => setVisible(true)} />
      <Dialog
        visible={visible}
        title="Confirm Action"
        message="Are you sure you want to proceed?"
        actions={[
          { label: 'Cancel', onPress: () => setVisible(false), variant: 'tertiary' },
          { label: 'Confirm', onPress: () => setVisible(false), variant: 'primary' },
        ]}
        onClose={() => setVisible(false)}
      />
    </>
  );
};

Props​

visible​

Type: boolean

Controls visibility of the dialog.

title​

Type: string

Optional title displayed at the top of the dialog.

message​

Type: string

Optional message text displayed in the dialog body. If children is provided, message is ignored.

children​

Type: React.ReactNode

Custom content to display in the dialog body. Takes precedence over message if both are provided.

actions​

Type: DialogAction[]

Array of action buttons to display at the bottom. Each action has:

  • label: string - Button text
  • onPress: () => void - Callback when button is pressed
  • variant?: 'primary' | 'secondary' | 'tertiary' - Button variant (default: 'primary')

If not provided, defaults to a single "OK" button that calls onClose.

showCloseButton​

Type: boolean

Default value: true

Whether to show the close icon button in the header.

onClose​

Type: () => void

Callback fired when dialog is closed (via backdrop, back button, or close icon).

backdropOpacity​

Type: number

Default value: 0.5

Opacity of the backdrop overlay (0-1).

animationIn​

Type: 'fadeIn' | 'slideInUp' | 'slideInDown' | 'zoomIn'

Default value: 'fadeIn'

Animation type when dialog appears.

animationOut​

Type: 'fadeOut' | 'slideOutDown' | 'slideOutUp' | 'zoomOut'

Default value: 'fadeOut'

Animation type when dialog disappears.

style​

Type: StyleProp<ViewStyle>

Custom styles for the dialog container.

containerStyle​

Type: StyleProp<ViewStyle>

Custom styles for the inner container.

titleStyle​

Type: StyleProp<ViewStyle>

Custom styles for the title text.

messageStyle​

Type: StyleProp<ViewStyle>

Custom styles for the message text.

actionsStyle​

Type: StyleProp<ViewStyle>

Custom styles for the actions container.

Code Examples​

import { Dialog, Button } from '@zaisustech/atomic-ui';
import { useState } from 'react';

function MyComponent() {
  const [visible, setVisible] = useState(false);

  return (
    <>
      <Button title="Show Dialog" onPress={() => setVisible(true)} />
      <Dialog
        visible={visible}
        title="Confirm Action"
        message="Are you sure you want to proceed?"
        actions={[
          { label: 'Cancel', onPress: () => setVisible(false), variant: 'tertiary' },
          { label: 'Confirm', onPress: () => setVisible(false), variant: 'primary' },
        ]}
        onClose={() => setVisible(false)}
      />
    </>
  );
}
// Dialog with custom content
<Dialog
  visible={isVisible}
  title="Custom Content"
  actions={[
    { label: 'OK', onPress: handleClose, variant: 'primary' },
  ]}
  onClose={handleClose}
>
  <Text>This is custom content instead of a message.</Text>
  <Button title="Action" onPress={handleAction} />
</Dialog>
// Dialog with different animations
<Dialog
  visible={isVisible}
  title="Animated Dialog"
  message="This dialog uses slide animations."
  animationIn="slideInUp"
  animationOut="slideOutDown"
  onClose={handleClose}
/>
