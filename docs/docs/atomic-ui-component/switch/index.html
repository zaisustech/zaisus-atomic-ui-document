Switch

A toggle switch component that allows users to turn an option on or off.

Category: Atom
Dependencies: Tokens (colors)

Interactive Preview​

Edit the code to customize the Switch:

Usage​

import { Switch } from '@zaisustech/atomic-ui';
import { useState } from 'react';

const MyComponent = () => {
  const [enabled, setEnabled] = useState(false);

  return (
    <Switch
      value={enabled}
      onValueChange={setEnabled}
      label="Enable notifications"
    />
  );
};

Props​

value​

Type: boolean

Required

The current value of the switch.

onValueChange​

Type: (value: boolean) => void

Required

Callback fired when the switch value changes. Receives the new boolean value.

disabled​

Type: boolean

Default value: false

Whether the switch is disabled.

label​

Type: string

Optional label text displayed next to the switch.

labelPosition​

Type: 'left' | 'right'

Default value: 'right'

Position of the label relative to the switch.

trackColor​

Type: { false?: string; true?: string }

Custom colors for the track when the switch is off (false) and on (true).

thumbColor​

Type: string

Custom color for the switch thumb.

ios_backgroundColor​

Type: string

Custom background color for iOS (only used on iOS).

testID​

Type: string

Optional test identifier for the component.

style​

Type: StyleProp<ViewStyle>

Custom styles for the container.

Examples​

Basic Switch​

import { Switch } from '@zaisustech/atomic-ui';

const [enabled, setEnabled] = useState(false);

<Switch value={enabled} onValueChange={setEnabled} />

With Label​

<Switch
  value={enabled}
  onValueChange={setEnabled}
  label="Enable notifications"
/>

<Switch
  value={enabled}
  onValueChange={setEnabled}
  label="Auto-save"
  labelPosition="left"
/>

Disabled State​

<Switch
  value={true}
  onValueChange={setEnabled}
  label="Disabled switch"
  disabled
/>

Custom Colors​

<Switch
  value={enabled}
  onValueChange={setEnabled}
  label="Custom colors"
  trackColor={{ false: '#E0E0E0', true: '#FF6B6B' }}
  thumbColor="#FFFFFF"
/>
