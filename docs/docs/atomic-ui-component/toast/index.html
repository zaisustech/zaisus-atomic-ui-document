Toast

A toast notification system using the useToast hook and ToastProvider.

Category: Molecule
Dependencies: Text, Icon, Button

  • success
  • error
  • warning
  • info

Interactive Preview​

Edit the code to customize the Toast:

Usage​

import { useToast, Button, ToastProvider } from '@zaisustech/atomic-ui';

function MyComponent() {
  const { showSuccessToast } = useToast();

  return (
    <Button 
      title="Show Success" 
      onPress={() => 
        showSuccessToast({
          title: 'Success!',
          message: 'Operation completed successfully.',
        })
      } 
    />
  );
}

function App() {
  return (
    <ToastProvider>
      <MyComponent />
    </ToastProvider>
  );
}

Setup​

First, wrap your app with ToastProvider:

import { ToastProvider } from '@zaisustech/atomic-ui';

function App() {
  return (
    <ToastProvider position="top">
      {/* Your app */}
    </ToastProvider>
  );
}

ToastProvider position​

Type: 'top' | 'middle' | 'bottom'
Default: 'top'

Where toasts appear on screen:

  • 'top' – Below the safe area at the top
  • 'middle' – Vertically centered
  • 'bottom' – Above the safe area at the bottom

useToast Hook​

The useToast hook provides methods to show different types of toasts:

MethodParametersDescription
showToast{ type: ToastType, title: string, message?: string, duration?: number }Show a toast with specified type
showSuccessToast{ title: string, message?: string, duration?: number }Show a success toast
showErrorToast{ title: string, message?: string, duration?: number }Show an error toast
showInfoToast{ title: string, message?: string, duration?: number }Show an info toast
showWarningToast{ title: string, message?: string, duration?: number }Show a warning toast

Toast Parameters​

title​

Type: string

Toast title text.

message​

Type: string

Optional message text below title.

duration​

Type: number

Default value: 3000

Duration in milliseconds before auto-dismiss.

Code Examples​

import { useToast, Button } from '@zaisustech/atomic-ui';

function MyComponent() {
  const { showSuccessToast, showErrorToast, showInfoToast, showWarningToast } = useToast();

  return (
    <>
      <Button 
        title="Show Success" 
        onPress={() => 
          showSuccessToast({
            title: 'Success!',
            message: 'Operation completed successfully.',
            duration: 3000
          })
        } 
      />
      <Button 
        title="Show Error" 
        onPress={() => 
          showErrorToast({
            title: 'Error',
            message: 'Something went wrong.'
          })
        } 
      />
    </>
  );
}
