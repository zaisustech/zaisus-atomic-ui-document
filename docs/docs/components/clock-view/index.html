ClockView

An interactive clock component for selecting hours and minutes with a real clock-like interface.

Category: Organism
Dependencies: SVG, Text, SegmentedControl, PeriodSelector, ClockFace

Interactive Preview​

Edit the code to customize the ClockView:

Usage​

import { ClockView } from '@zaisustech/atomic-ui';
import { useState } from 'react';

const MyComponent = () => {
  const [time, setTime] = useState({
    hour: 3,
    minute: 45,
  });

  return <ClockView value={time} onSelect={setTime} />;
};

Props​

value​

Type: { hour: number; minute: number; second?: number }

Default value: undefined

The current time value. The hour should be in 12-hour format (1-12). Seconds are optional (0-59).

onSelect​

Type: (time: { hour: number; minute: number; second?: number }) => void

Required. Callback fired when time is selected. The callback receives the selected time object with hour, minute, and optional second properties.

size​

Type: number

Default value: 300

Size of the clock face in pixels. Controls both width and height of the clock component.

backgroundColor​

Type: string

Default value: undefined (uses theme colors.surface)

Custom background color for the clock face circle.

handColor​

Type: string

Default value: undefined (uses theme colors.primary)

Custom color for the hour and minute hands, as well as the center dot.

selectedHourColor​

Type: string

Default value: undefined (uses theme colors.primary)

Custom color for the selected hour number.

numberColor​

Type: string

Default value: undefined (uses theme colors.text)

Custom color for non-selected hour numbers.

numberSize​

Type: number

Default value: undefined (16 for unselected, 18 for selected)

Base font size for hour numbers in pixels. Selected hour numbers are automatically 2px larger.

markerColor​

Type: string

Default value: undefined (uses theme colors.text)

Custom color for hour and minute markers (the tick marks on the clock face).

viewMode​

Type: 'default' | 'hour' | 'minute'

Default value: undefined (internal state, defaults to 'default')

The current view mode of the clock. When undefined, the clock manages view mode internally (automatically switches to minute mode after hour selection). When provided, the clock becomes controlled and requires onViewModeChange to update the mode.

  • 'default' - Initial state, allows hour selection
  • 'hour' - Hour selection mode
  • 'minute' - Minute selection mode

onViewModeChange​

Type: (mode: 'default' | 'hour' | 'minute') => void

Default value: undefined

Callback fired when the view mode changes. Use this to control the view mode externally, typically with a SegmentedControl or similar component.

showTick​

Type: boolean

Default value: false

Whether to show the second hand (tick) on the clock. When true, the second hand continuously animates using the current system time.

Example​

Basic ClockView​

import { ClockView } from '@zaisustech/atomic-ui';
import { useState } from 'react';
import { View } from 'react-native';

function MyComponent() {
  const [time, setTime] = useState({
    hour: 3,
    minute: 45,
  });

  return (
    <View style={{ alignItems: 'center', padding: 20 }}>
      <ClockView value={time} onSelect={setTime} />
    </View>
  );
}

Custom Size​

import { ClockView } from '@zaisustech/atomic-ui';
import { useState } from 'react';
import { View } from 'react-native';

function MyComponent() {
  const [time, setTime] = useState({
    hour: new Date().getHours() % 12 || 12,
    minute: new Date().getMinutes(),
  });

  return (
    <View style={{ alignItems: 'center', padding: 20 }}>
      <ClockView value={time} onSelect={setTime} size={250} />
    </View>
  );
}

Custom Colors and Styling​

import { ClockView } from '@zaisustech/atomic-ui';
import { useState } from 'react';
import { View } from 'react-native';

function MyComponent() {
  const [time, setTime] = useState({
    hour: 3,
    minute: 45,
  });

  return (
    <View style={{ alignItems: 'center', padding: 20 }}>
      <ClockView
        value={time}
        onSelect={setTime}
        backgroundColor="#F0F0F0"
        handColor="#FF6B6B"
        selectedHourColor="#4ECDC4"
        numberColor="#666666"
        numberSize={20}
        markerColor="#333333"
      />
    </View>
  );
}

Behavior​

  • Hour Selection: Tap or drag on the clock face to select an hour. The hour hand will move to the selected position.
  • Minute Selection: After selecting an hour, the view automatically switches to minute selection mode. The minute hand appears and you can select minutes.
  • Second Hand: The second hand (optional, when showTick is true) continuously animates using the current system time. It's more visible in minute selection mode.
  • Back Navigation: When in minute selection mode, a "Back to Hour" button appears to return to hour selection.

Design​

The clock features:

  • 12-hour format with numbers 1-12 displayed around the face
  • Hour markers (12 major markers) and minute markers (60 minor markers, visible in minute mode)
  • Hour hand (thicker, primary color) and minute hand (thinner, primary color)
  • Second hand (red, continuously animating, optional via showTick prop)
  • Touch interaction - tap or drag to select hours/minutes

Dependencies​

  • react-native-svg - For rendering the clock face and hands
