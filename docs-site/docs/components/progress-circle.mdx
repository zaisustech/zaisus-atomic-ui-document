# ProgressCircle

A circular progress indicator component with animation support.

## Live Example

Try the interactive examples below:

<ProgressCirclePreview showMultiple={true} />

## Usage

```tsx
import { ProgressCircle } from '@zaisustech/atomic-ui';

const MyComponent = () => (
  <ProgressCircle progress={75} />
);
```

## Props

### progress

Type: `number`

Progress value from 0 to 100.

### size

Type: `number`

Default value: `100`

Diameter of the circle in pixels.

### strokeWidth

Type: `number`

Default value: `8`

Width of the stroke in pixels.

### color

Type: `string`

Color of the progress stroke.

### backgroundColor

Type: `string`

Color of the background circle.

### showLabel

Type: `boolean`

Default value: `false`

Whether to display percentage label in center.

### labelStyle

Type: `StyleProp<ViewStyle>`

Custom styles for the label container.

### style

Type: `StyleProp<ViewStyle>`

Custom styles for the container.

### animated

Type: `boolean`

Default value: `true`

Whether progress changes are animated.
