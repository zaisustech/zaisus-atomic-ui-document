# Badge

A small component used to display labels, counts, or status indicators.

* default
* primary
* secondary
* success
* error
* warning

## Live Example

Try the interactive examples below:

<BadgePreview showMultiple={true} />

## Usage

```tsx
import { Badge } from '@zaisustech/atomic-ui';

const MyComponent = () => (
  <Badge label="New" variant="primary" size="medium" />
);
```

## Props

### label

Type: `string | number`

Text or number to display in the badge.

### variant

Type: `'default' | 'primary' | 'secondary' | 'success' | 'error' | 'warning'`

Default value: `'default'`

Visual style variant. You can change the variant to adjust the styling and semantic meaning.

* `'default'` - Standard badge with neutral styling
* `'primary'` - Primary brand color variant
* `'secondary'` - Secondary brand color variant
* `'success'` - Green color variant for success states
* `'error'` - Red color variant for error states
* `'warning'` - Orange/yellow color variant for warning states

### size

Type: `'small' | 'medium' | 'large'`

Default value: `'medium'`

Size of the badge. Controls the overall dimensions and padding of the badge.

### style

Type: `StyleProp<ViewStyle>`

Custom styles for the container.

### textStyle

Type: `StyleProp<ViewStyle>`

Custom styles for the text.
