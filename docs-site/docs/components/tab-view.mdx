# TabView

A tab view component for displaying multiple tabs with an animated indicator.

* fixed
* scroll

## Live Example

Try the interactive examples below:

<TabViewPreview showMultiple={true} />

## Usage

```tsx
import { TabView, Text } from '@zaisustech/atomic-ui';
import { useState } from 'react';

const MyComponent = () => {
  const [selectedTab, setSelectedTab] = useState(0);

  return (
    <TabView type="scroll" value={selectedTab} onChange={setSelectedTab}>
      <View><Text>Tab 1</Text></View>
      <View><Text>Tab 2</Text></View>
      <View><Text>Tab 3</Text></View>
    </TabView>
  );
};
```

## Props

### children

Type: `React.ReactNode`

Tab elements to display.

### value

Type: `number`

Currently selected tab index.

### onChange

Type: `(page: number) => void`

Callback fired when tab is selected.

### type

Type: `'fixed' | 'scroll'`

Default value: `'scroll'`

Tab layout type. Controls how tabs are distributed and whether they can scroll.

* `'fixed'` - Tabs are distributed evenly across the available width
* `'scroll'` - Tabs can scroll horizontally when they exceed the available width

### style

Type: `StyleProp<ViewStyle>`

Custom styles for the container.

### containerStyle

Type: `StyleProp<ViewStyle>`

Custom styles for the content container.

### selectionStyle

Type: `StyleProp<ViewStyle>`

Custom styles for the selection indicator.
