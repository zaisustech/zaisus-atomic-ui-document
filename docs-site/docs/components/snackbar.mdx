# Snackbar

A temporary notification component that appears at the bottom of the screen.

* default
* success
* error
* warning
* info

## Live Example

Try the interactive examples below:

<SnackbarPreview showMultiple={true} />

## Usage

```tsx
import { Snackbar } from '@zaisustech/atomic-ui';

const MyComponent = () => {
  const [visible, setVisible] = useState(false);

  return (
    <Snackbar
      visible={visible}
      message="Operation completed successfully"
      type="success"
      duration={4000}
      onDismiss={() => setVisible(false)}
    />
  );
};
```

## Props

### visible

Type: `boolean`

Default value: `false`

Controls visibility of the snackbar.

### message

Type: `string`

Message text to display.

### type

Type: `'default' | 'success' | 'error' | 'warning' | 'info'`

Default value: `'default'`

Visual type of the snackbar. You can change the type to adjust the styling and semantic meaning.

* `'default'` - Standard snackbar with neutral styling
* `'success'` - Green color variant for success messages
* `'error'` - Red color variant for error messages
* `'warning'` - Orange/yellow color variant for warning messages
* `'info'` - Blue color variant for informational messages

### duration

Type: `number`

Default value: `4000`

Duration in milliseconds before auto-dismiss.

### actionLabel

Type: `string`

Label for the action button.

### onActionPress

Type: `() => void`

Callback fired when action button is pressed.

### onDismiss

Type: `() => void`

Callback fired when snackbar is dismissed.

### style

Type: `StyleProp<ViewStyle>`

Custom styles for the container.

### Code Examples

```tsx
import { Snackbar } from '@zaisustech/atomic-ui';
import { useState } from 'react';

function MyComponent() {
  const [visible, setVisible] = useState(false);

  return (
    <>
      <Button title="Show Snackbar" onPress={() => setVisible(true)} />
      <Snackbar
        visible={visible}
        message="Operation completed successfully"
        type="success"
        duration={4000}
        actionLabel="Undo"
        onActionPress={() => console.log('Undo pressed')}
        onDismiss={() => setVisible(false)}
      />
    </>
  );
}
```
