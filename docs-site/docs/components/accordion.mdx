# Accordion

A collapsible content component that expands and collapses with animation.

## Live Example

Try the interactive examples below:

<AccordionPreview showMultiple={true} />

## Usage

```tsx
import { Accordion, Text } from '@zaisustech/atomic-ui';

const MyComponent = () => (
  <Accordion 
    title="Click to expand"
    defaultExpanded={false}
    onToggle={(expanded) => console.log('Expanded:', expanded)}
  >
    <Text variant="Paragraph.default.regular">
      This content appears when expanded.
    </Text>
  </Accordion>
);
```

## Props

### title

Type: `string`

The header text displayed in the accordion.

### children

Type: `React.ReactNode`

Content to display when expanded.

### defaultExpanded

Type: `boolean`

Default value: `false`

Whether the accordion starts expanded.

### style

Type: `StyleProp<ViewStyle>`

Custom styles for the container.

### headerStyle

Type: `StyleProp<ViewStyle>`

Custom styles for the header.

### contentStyle

Type: `StyleProp<ViewStyle>`

Custom styles for the content area.

### onToggle

Type: `(expanded: boolean) => void`

Callback fired when accordion is toggled.

### Code Examples

```tsx
import { Accordion, Text } from '@zaisustech/atomic-ui';

// Basic accordion
<Accordion 
  title="Click to expand"
  defaultExpanded={false}
  onToggle={(expanded) => console.log('Expanded:', expanded)}
>
  <Text variant="Paragraph.default.regular">
    This content appears when expanded.
  </Text>
</Accordion>

// Accordion that starts expanded
<Accordion 
  title="Starts Expanded"
  defaultExpanded={true}
  onToggle={(expanded) => console.log('Expanded:', expanded)}
>
  <Text variant="Paragraph.default.regular">
    This accordion is expanded by default.
  </Text>
</Accordion>
```
