# Divider

A visual separator component that can be horizontal or vertical.

* horizontal
* vertical

## Live Example

Try the interactive examples below:

<DividerPreview showMultiple={true} />

## Usage

```tsx
import { Divider } from '@zaisustech/atomic-ui';

const MyComponent = () => (
  <>
    <Text>Content above</Text>
    <Divider orientation="horizontal" />
    <Text>Content below</Text>
  </>
);
```

## Props

### orientation

Type: `'horizontal' | 'vertical'`

Default value: `'horizontal'`

Direction of the divider. Controls whether the divider is displayed horizontally or vertically.

* `'horizontal'` - Divider extends horizontally (default)
* `'vertical'` - Divider extends vertically

### style

Type: `StyleProp<ViewStyle>`

Custom styles for the divider.

```tsx
import { Divider } from '@zaisustech/atomic-ui';

// Horizontal divider
<Text>Content above</Text>
<Divider orientation="horizontal" />
<Text>Content below</Text>

// Vertical divider
<View style={{ flexDirection: 'row' }}>
  <Text>Left</Text>
  <Divider orientation="vertical" />
  <Text>Right</Text>
</View>
```
