# Text

A text component with predefined typography variants.

## Live Example

Try the interactive examples below:

<TextPreview showMultiple={true} />

## Usage

```tsx
import { Text } from '@zaisustech/atomic-ui';

const MyComponent = () => (
  <Text variant="Header.large.bold">Main Title</Text>
);
```

## Props

### children

Type: `string | React.ReactNode | React.ReactNode[]`

Text content or React nodes.

### value

Type: `string`

Alternative text value (for i18n).

### variant

Type: `TextVariant`

Default value: `'Paragraph.default.regular'`

Typography variant. You can change the variant to adjust the font size, weight, and style. See variants below for all available options.

### style

Type: `StyleProp<TextStyle> | StyleProp<TextStyle>[]`

Custom styles for the text.

## Text Variants

### Paragraph
- `'Paragraph.default.regular'`, `'Paragraph.small.regular'`, `'Paragraph.large.regular'`
- `'Paragraph.default.bold'`, `'Paragraph.small.bold'`, `'Paragraph.large.bold'`
- `'Paragraph.default.italic'`, `'Paragraph.small.italic'`, `'Paragraph.large.italic'`

### Header
- `'Header.default.regular'`, `'Header.small.regular'`, `'Header.large.regular'`
- `'Header.default.bold'`, `'Header.small.bold'`, `'Header.large.bold'`
- `'Header.default.italic'`, `'Header.small.italic'`, `'Header.large.italic'`

### Label
- `'Label.regular'`, `'Label.bold'`, `'Label.italic'`, `'Label.small'`
