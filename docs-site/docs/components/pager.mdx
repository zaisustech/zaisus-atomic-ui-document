# Pager

A pager component for displaying multiple pages that can be swiped horizontally or switched programmatically.

* flat
* view

## Live Example

Try the interactive examples below:

<PagerPreview showMultiple={true} />

## Usage

```tsx
import { Pager, Text } from '@zaisustech/atomic-ui';
import { useState } from 'react';

const MyComponent = () => {
  const [currentPage, setCurrentPage] = useState(0);

  return (
    <Pager type="flat" page={currentPage} onPageChange={setCurrentPage} scrollEnabled>
      <View><Text>Page 1</Text></View>
      <View><Text>Page 2</Text></View>
      <View><Text>Page 3</Text></View>
    </Pager>
  );
};
```

## Props

### children

Type: `React.ReactNode[]`

Array of page elements to display.

### page

Type: `number`

Currently active page index.

### onPageChange

Type: `(index: number) => void`

Callback fired when page changes.

### type

Type: `'flat' | 'view'`

Default value: `'flat'`

Pager type. Controls the underlying implementation and behavior of the pager.

* `'flat'` - Uses FlatList for efficient scrolling and rendering of pages
* `'view'` - Uses View for programmatic switching without scrolling

### scrollEnabled

Type: `boolean`

Default value: `false`

Whether scrolling is enabled (only applies to 'flat' type).

### Code Examples

```tsx
import { Pager, Text } from '@zaisustech/atomic-ui';
import { useState } from 'react';

function MyComponent() {
  const [currentPage, setCurrentPage] = useState(0);

  return (
    // FlatList type with scrolling
    <Pager type="flat" page={currentPage} onPageChange={setCurrentPage} scrollEnabled>
      <View><Text>Page 1</Text></View>
      <View><Text>Page 2</Text></View>
      <View><Text>Page 3</Text></View>
    </Pager>
  );
}

// View type for programmatic switching
<Pager type="view" page={currentPage} onPageChange={setCurrentPage}>
  <View><Text>Page 1</Text></View>
  <View><Text>Page 2</Text></View>
  <View><Text>Page 3</Text></View>
</Pager>
```
