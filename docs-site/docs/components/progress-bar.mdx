# ProgressBar

A horizontal progress bar component with animation support.

## Live Example

Try the interactive examples below:

<ProgressBarPreview showMultiple={true} />

## Usage

```tsx
import { ProgressBar } from '@zaisustech/atomic-ui';

const MyComponent = () => (
  <ProgressBar progress={75} />
);
```

## Props

### progress

Type: `number`

Progress value from 0 to 100.

### height

Type: `number`

Default value: `8`

Height of the progress bar in pixels.

### color

Type: `string`

Color of the progress fill.

### backgroundColor

Type: `string`

Color of the progress track background.

### showLabel

Type: `boolean`

Default value: `false`

Whether to display percentage label.

### labelStyle

Type: `StyleProp<ViewStyle>`

Custom styles for the label container.

### style

Type: `StyleProp<ViewStyle>`

Custom styles for the container.

### animated

Type: `boolean`

Default value: `true`

Whether progress changes are animated.
